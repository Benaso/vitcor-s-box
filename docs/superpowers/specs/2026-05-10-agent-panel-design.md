# Agent 面板设计文档

## 概述

点击首页"嗨！马文！"按钮后，页面分屏显示 - 左侧主页内容收缩，右侧新增 agent 面板区域，包含启动序列动画，然后展开为可交互的 agent 对话栏。

## 交互流程

1. 点击"嗨！马文！"按钮
2. 页面分屏动画（300ms ease-out）- 左侧主页收缩至约 60%，右侧出现 agent 面板（约 40%）
3. 中间有炭灰色分隔线（4px）
4. 右侧面板内显示启动序列，逐行加载
5. 启动完成后，隐藏启动序列，显示 agent 对话界面
6. 点击关闭按钮，左侧主页恢复全宽，右侧面板消失

## 布局规格

### 分屏比例
- **桌面 (>= 1024px)**: 左侧 60%，右侧 40%，中间分隔线 4px
- **平板/手机 (< 1024px)**: 上方 50% 主页，下方 50% agent 面板

### 面板尺寸
- **高度**: `100vh - 导航栏高度`，从导航栏下方开始
- **宽度**: 右侧 40%（桌面）/ 100%（移动端）

### 分隔线
- 宽度: 4px
- 颜色: #2a2a2a（炭灰色）
- 样式: 实线

### 响应式
- 桌面: 左右分屏
- 平板/手机: 上下分屏

## 启动序列动画

逐行显示以下内容，每行动画间隔 300ms：

```
> initializing marvin module...       [完成]
> loading personality matrix...       [完成]
> establishing neural link...         [完成]
> calibrating empathic processors...  [完成]
> BT-7274 protocol online            [完成]
> Marvin ready.
```

- 每行从左到右打字机效果
- 行尾显示 `[完成]` 标记
- 最后一行完成后，停留 500ms，然后切换到对话界面

## 对话界面

### 结构
```
┌─────────────────────────────┐
│ ● ● ●  marvin.agent     [X] │  <- 标题栏
├─────────────────────────────┤
│                             │
│  [消息列表区域]              │  <- 可滚动
│                             │
├─────────────────────────────┤
│ $ [输入框]              [发送]│  <- 输入区
└─────────────────────────────┘
```

### 标题栏
- 三个圆点（装饰）
- 标题 "marvin.agent"
- 关闭按钮 [X]

### 消息列表
- 用户消息和 Marvin 消息交替显示
- 用户消息右对齐，灰色背景
- Marvin 消息左对齐，白色背景
- 支持 markdown 渲染

### 输入区
- `$` 提示符
- 文本输入框
- 发送按钮

## 技术实现

### 组件结构
```
BTAgentPanel (主组件)
├── BTAgentLauncher (触发按钮，首页内)
├── BTAgentSidebar (右侧分屏面板)
│   ├── BTBootSequence (启动序列)
│   └── BTChatInterface (对话界面)
```

### Context
- `BTTerminalContext` - 控制面板开关状态

### 样式
- 使用 CSS flexbox 实现分屏布局
- 使用 CSS transitions 实现分屏动画
- 使用 CSS animations 实现打字机效果

## 验收标准

1. 点击按钮后页面分屏（左侧 60%，右侧 40%）
2. 中间有 4px 分隔线
3. 面板高度在导航栏下方
4. 启动序列逐行动画显示
5. 启动完成后切换到对话界面
6. 可发送消息并接收回复
7. 可关闭面板并恢复全宽
8. 移动端自适应（上下分屏）
