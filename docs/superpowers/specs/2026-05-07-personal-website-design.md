# 个人网站设计文档

**日期**: 2026-05-07
**状态**: 已批准

---

## 概述

个人网站，用于展示个人爱好、项目作品、碎碎念、成长经历和职业规划。采用 React + Rust 后端架构，设计风格为柔和黑白像素风。

---

## 设计理念

### 视觉风格：柔和黑白像素风

**配色方案**:
- 米白底 `#f5f2eb` — 温暖柔和的背景
- 炭灰边框 `#2a2a2a` — 强对比但不过刺眼
- 纯黑白灰 — 无其他颜色点缀

**像素元素**:
- 4px 粗边框
- 固体阴影 (4-8px offset)
- 等宽字体 (monospace)
- 8-bit 风格标题字体

**布局原则**:
- 网格化设计
- 卡片式组件
- 大量留白
- 清晰的视觉层次

## 页面结构

| 页面 | 路由 | 内容 |
|------|------|------|
| 首页 | `/` | 简介 + 核心技能状态卡 |
| 关于 | `/about` | 成长经历时间线 |
| 项目 | `/projects` | 作品展示卡片 |
| 职业规划 | `/career` | 未来发展方向 |
| 碎碎念 | `/blog` | 博客/日志列表 |
| 爱好 | `/hobbies` | 个人兴趣展示 |

---

## 技术架构

### 前端
- React 18 + Vite
- React Router v6
- 纯 CSS（无框架）

### 后端
- Rust + Axum
- JSON 文件存储
- RESTful API

### API 设计

| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/profile` | GET | 获取个人简介 |
| `/api/projects` | GET | 获取项目列表 |
| `/api/blog` | GET | 获取碎碎念列表 |
| `/api/career` | GET | 获取职业规划 |
| `/api/hobbies` | GET | 获取爱好列表 |

## 响应式设计

| 设备 | 断点 | 布局 |
|------|------|------|
| 桌面 | ≥1024px | 双栏/多栏 |
| 平板 | 768px-1023px | 单栏为主 |
| 手机 | <768px | 全宽单栏 |
