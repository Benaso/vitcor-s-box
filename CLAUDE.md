# 个人网站项目

## 约束

- **中文交流** — 所有沟通使用中文

## GitHub 仓库

- **仓库地址**: https://github.com/Benaso/vitcor-s-box
- **部署方式**: GitHub Pages (通过 Actions 自动部署)

## 设计理念

### 视觉风格：柔和黑白像素风

```
配色方案：
- 米白底 (#f5f2eb) — 温暖柔和的背景
- 炭灰边框 (#2a2a2a) — 强对比但不过刺眼
- 纯黑白灰 — 无其他颜色点缀

像素元素：
- 4px 粗边框
- 固体阴影 (4-8px offset)
- 等宽字体 (monospace)
- 8-bit 风格标题字体

布局：
- 网格化设计
- 卡片式组件
- 大量留白
```

### 页面结构

| 板块 | 内容 |
|------|------|
| 首页 | 简介 + 核心技能状态卡 |
| 关于 | 成长经历 |
| 项目 | 作品展示 |
| 职业规划 | 未来方向 |
| 碎碎念 | 博客/日志 |
| 爱好 | 个人兴趣 |

### 技术栈

- **前端**: React + Vite
- **数据**: JSON 文件直接引入
- **部署**: GitHub Pages (静态托管)

### 部署说明

1. 前端构建: `cd frontend && pnpm build`
2. 构建产物在 `frontend/dist/`
3. 部署到 GitHub Pages (通过 GitHub Actions 或手动上传)

### 响应式断点

- 桌面: 1024px+
- 平板: 768px - 1023px
- 手机: < 768px
